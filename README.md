# Expense Tracker

A modern and responsive Expense Tracker web application built with React and Tailwind CSS. This application helps users manage their daily expenses by tracking income, expenses, and overall balance in a clean and user-friendly interface.

## Live Demo

🔗 https://parthivjain.github.io/Expance_Tracker_React_Copy1/

## Features

* Add income and expense transactions
* Track total balance
* View complete transaction history
* Responsive design for mobile and desktop devices
* Clean and modern user interface
* Fast and lightweight React application
* Built using reusable React components

## Tech Stack

### Frontend

* React
* JavaScript (ES6+)
* Tailwind CSS
* Vite

### Deployment

* GitHub Pages
